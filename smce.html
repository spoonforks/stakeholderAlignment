SMCE process adapted from SOCRATES

Step 1
Frame alternatives

Define competing erfpacht governance packages instead of evaluating one preferred model in isolation.

Step 2
Structure dimensions

Translate post-growth aims into explicit dimensions, objectives, and criteria.

Step 3
Build the impact matrix

Score each alternative against each criterion using a common 1-5 impact scale.

Step 4
Technical ranking

Use weighted pairwise criterion wins and a Kemeny-style compromise ranking rather than a simple fully compensatory average.

Step 5
Equity analysis

Compare how stakeholder groups rate the alternatives and detect coalition proximity or conflict.

Step 6
Sensitivity and robustness

Test exclusions, boosted priorities, and random weight combinations before using the ranking in negotiation.

Case fit

1. Agreement alternatives

2. Dimensions, criteria, and impact matrix

Dimension weights shape the macro-political priorities. Criterion weights shape how each dimension is operationalised.

3. Stakeholder equity matrix

Each actor rates how acceptable each alternative is for them. Actor weights express representation importance in the negotiation table.

4. Technical ranking

Non-compensatory ranking from weighted pairwise criterion wins.

5. Equity ranking

Social compromise ranking from weighted stakeholder preferences.

6. Recommendation logic

The agreement pathway should survive both technical and social scrutiny.
