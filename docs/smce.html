Method note: this page reconstructs the core SOCRATES / SMCE workflow in a workshop-friendly static web tool. It uses weighted pairwise comparisons with a Kemeny-style ranking search to avoid a purely compensatory average, then adds stakeholder equity and robustness checks for the Marineterrein post-growth erfpacht case. Official references: European Commission SOCRATES, JRC model inventory, JRC manual on social multi-criteria evaluation.
